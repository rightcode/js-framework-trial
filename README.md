# js-frameworks-trial

# Trial
## AngularJS
AngularJSを試す場合は、`angular-trial`ディレクトリへ移動して、以下のコマンドを実行してください。

```shell script
npm install
ng serve --open
```

## React
Reactを試す場合は、`react-trial`ディレクトリへ移動して、以下のコマンドを実行してください。

```shell script
npm install
npm start
```

## Vue.js
Vue.jsを試す場合は、`vue-trial`ディレクトリへ移動して、以下のコマンドを実行してください。

```shell script
npm install
npm run serve
```
